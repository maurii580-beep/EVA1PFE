Pixel Fotografías - Landing Page Profesional
Descripción del Proyecto
Este proyecto consiste en el desarrollo de una Landing Page funcional para un emprendimiento de fotografía.

Tecnologías Utilizadas
HTML5: Estructura semántica completa utilizando etiquetas como <header>, <nav>, <main>, <section>, <article> y <footer>.

CSS3: Estilos personalizados para el manejo de colores, tipografías, márgenes y rellenos.

Bootstrap 5: Framework CSS utilizado para el sistema de rejilla (Grid) responsivo y componentes como tarjetas y formularios.

Git & GitHub: Gestión de versiones con un historial de commits descriptivos para respaldar el avance del proyecto.

Uso de Inteligencia Artificial
Se integró el uso de IA para optimizar el desarrollo de la siguiente manera:

Generación de código: Apoyo en la creación de la estructura de la galería responsiva y el formulario de contacto.

Depuración y Optimización: Resolución de errores en rutas de imágenes y configuración de comandos Git para el despliegue.
